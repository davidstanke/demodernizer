# Bakery website
A simple, one-page website for a business called **Legacy Bakery**.

## Requirements
- **Main Heading:** Must display "Legacy Bakery" as an H1 heading.
- **Menu:** Must list at least the following products:
    - Classic Croissant
    - Sourdough Loaf
    - Pain au Chocolat
- **Hours Section:**
    - Must have a heading titled "Hours".
    - Must display "Monday - Friday:".
- **Contact Information:**
    - Must include the address "123 Bakery Lane" and "Jersey City, NJ 07302".
